2. Prender LED por 5 minutos
3. IVg prendido
4. IVg apagado de inmediato y obtener DP [V]
    + En caso de haber dos DP, usar el promedio
5. It para Vg = DP + Delta elegido
6. IVg apagado
7. Esperar 30 minutos
8. IVg apagado
